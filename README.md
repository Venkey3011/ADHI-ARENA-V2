# ADHI ARENA 🚀

**ADHI ARENA** is a desktop application developed for conducting **Coding and Aptitude Tests** for students.

It provides a simple platform for administrators to create tests, manage students, conduct coding contests, and view results.

---

## ✨ Features

* Conduct Coding Tests
* Conduct Aptitude Tests
* Create Coding Contests
* Manage Students
* Bulk Upload Students using Excel
* Coding and Aptitude Question Banks
* Automatic Code Evaluation
* Multiple Test Case Validation
* Hidden Test Cases
* Student Result Management
* Contest Timer
* Local Compiler Execution
* Automatic Application Updates

---

## 💻 Supported Programming Languages

* Java
* Python
* C
* C++
* JavaScript

The application automatically detects the compilers installed on the student's computer.

Coding programs are compiled and executed directly on the local system.

## 📥 Installation

Clone the repository:

```bash
git clone https://github.com/Venkey3011/ADHI-ARENA-V2.git
```

Open the project folder:

```bash
cd ADHI-ARENA-V2
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Run the Application

Start the development server:

```bash
npm run dev
```

Run the desktop application:

```bash
npm run desktop
```

---

## 📦 Build Windows Installer

```bash
npm run installer
```

The Windows installer will be generated inside the `release` folder.
